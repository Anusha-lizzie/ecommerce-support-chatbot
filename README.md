# E-commerce Customer Support Chatbot

An AI-powered customer support chatbot built using Python, Streamlit, and Gemini.
Uses Retrieval-Augmented Generation (RAG) to answer queries based on store policies.

## Features
- AI-based chat interface
- Policy-aware answers using RAG
- Clean Streamlit UI

## Tech Stack
- Python
- Streamlit
- Gemini LLM
- FAISS
